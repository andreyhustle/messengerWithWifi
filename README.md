# messengerWithWifi
Creating a messenger to chat over Internet and Wi-fi when Internet is turned down

<h2>Описание идеи продукта</h2>

Приложение MWW предназначено для общения пользователей с помощью интернета используя номер своего телефона. <br>
<h2>Основные функции</h2> <br>
<ul> <li>возможность создавать групповые чаты </li>
  <li>возможность переписки с человеком тет-а-тет</li>
  <li>возможность отправки человеку файлов, изображений.</li>
  <li>возможность блокировки пользователей</li>
  <li>настройка личного аккаунта(Имя, фото, возраст, личная информация и др.)</li>
  <li>возможность удаления чатов или переписок тет-а-тет</li> </ul>
  <h2>Технические требования</h2>
 <ul><li>iPhone 6,6+,6s,6s+,7,7+,8,8+</li>
  <li>доступ к интернету</li>
  <li>доступ к файловой системе</li></ul>
<h2>Описание потенциальной аудитории</h2>
Приложение будет создано с целью общения людей друг с другом через интернет.
